沒啦，這是一個用來編輯WARFRAME裡三線琴的琴譜的網站~
## 建議解析度
Windows縮放100%，1920*1080的Chrome瀏覽器
## 如何使用
本網站有兩種使用方式，一種是透過紅底的指標並按下表格上的字母來輸入
也可以直接在表格上用鍵盤打，主要希望手機也可以編輯，
~~雖然用手機看這網頁是醜不拉嘰就是了~~
## 注意事項
* **本網站並不會暫存你的編曲**
所以每次重新整理和匯入都會把編曲表格清空，敬請注意
不要把辛苦編來的曲子丟掉了!
* 匯入的功能尚不完善，還請在匯入前確保歌曲的格式正確無誤!
